# 🚀 Web'in 3 Atlısı: HTML5, CSS3 ve Flask

Bu proje, modern web geliştirmenin temel yapı taşları olan **HTML5**, **CSS3** ve **Flask** teknolojilerinin özelliklerini interaktif, animasyonlu ve sesli bir şekilde anlatan tek sayfalık bir web sunumudur. 

## ✨ Özellikler

- **Bağımlılık Yok:** Sadece saf HTML, CSS ve JavaScript (Vanilla JS) ile yazılmıştır. Herhangi bir kütüphane veya Node.js kurulumu gerektirmez.
- **Dinamik Animasyonlar:** CSS3 `keyframes` ve `transition` özellikleri kullanılarak modern bir görsellik sunar.
- **Yapay Zeka Seslendirme:** Tarayıcının yerleşik **Web Speech API** özelliği sayesinde, her slayta geçişinde otomatik Türkçe seslendirme yapar.
- **Tarayıcı İçi Ekran Kaydı:** `MediaRecorder` API kullanılarak, sunum oynatılırken arka planda video olarak kaydedilebilir. Sunum bittiğinde video doğrudan **MP4 / WebM** formatında bilgisayarınıza indirilir.

## 🛠️ Nasıl Kullanılır?

Projeyi çalıştırmak ve sunumu izlemek çok basittir:

1. Bu depoyu (repository) bilgisayarınıza indirin veya klonlayın.
2. `index.html` dosyasını favori modern tarayıcınızda (Google Chrome, Firefox, Edge, Safari vb.) açın.
3. Karşınıza çıkan ekrandaki butonları kullanın:
   - **▶️ Sunumu ve Seslendirmeyi Başlat:** Sunumu sadece izlemek ve dinlemek istiyorsanız tıklayın.
   - **⏺️ Ekranı Kaydet (MP4/WebM İndir):** Sunumu video olarak kaydetmek istiyorsanız tıklayın. Tarayıcı ekran paylaşımı isteyecek; sunumun açık olduğu sekmeyi seçmeniz yeterlidir. Sunum bittiğinde video dosyası otomatik olarak inecektir.

## 🌍 GitHub Pages ile Canlı Yayın (Deploy)

Bu proje GitHub Pages ile %100 uyumludur ve ana sayfa `index.html` olarak ayarlanmıştır.
Deposunu kendi hesabınıza *fork* ettikten veya oluşturduktan sonra:
1. GitHub deponuzdan `Settings` (Ayarlar) sekmesine gidin.
2. Sol menüden `Pages` bölümünü seçin.
3. **Source** kısmını `Deploy from a branch` olarak seçin.
4. Branch olarak `main` (veya `master`) seçip kaydedin.
5. Birkaç dakika içinde projeniz canlı bir web sitesi olarak yayınlanacaktır!
